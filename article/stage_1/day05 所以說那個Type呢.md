有哪些Type 大概使用場景

JavaScript 的型別分為兩種：
1. 原始資料型別(Primitive data types) => {
    boolean (布林值),
    number (數值),
    string (字串),
    null,
    undefined,
    Symbol(ES6),
}
2. 物件型別(Object types) => {
    object(物件),
    array(陣列),
    function(函式)
}
